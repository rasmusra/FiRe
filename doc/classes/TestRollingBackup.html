Class TestRollingBackup
In: test/full/tc_rolling_backup.rb
Parent: Test::Unit::TestCase

The Rolling-Backup worker moves given destination to a backup- folder below the root of backup-folder. The paramerer "history" states how many backups will be kept before overwriting the oldest. Please note that this worker does not copy anything, only moves backup-dir to an archive-folder. The archive is named "rolling backup yyyy-mm-dd HHMMSS".
